# Figma 디자인 반영 가이드

Figma 디자인을 코드로 반영하는 방법입니다.

## Figma에서 디자인 정보 확인하기

### 방법 1: Figma Dev Mode 사용 (추천)

1. Figma에서 디자인 파일 열기
2. 우측 상단 "Dev Mode" 토글 활성화
3. 요소를 선택하면 오른쪽에 CSS 정보 표시
4. 다음 정보를 확인:
   - **Padding/Margin**: 간격 정보
   - **Colors**: 색상 코드 (hex, rgb)
   - **Typography**: 폰트 크기, 굵기, 행간
   - **Spacing**: 요소 간 간격

### 방법 2: Inspect 패널 사용

1. 요소 선택
2. 우측 패널에서 "Inspect" 탭 클릭
3. CSS 속성 확인

### 방법 3: 수동으로 측정

1. 요소 선택
2. 좌측 상단에 크기 정보 표시 (width, height)
3. 요소 간 간격은 두 요소를 선택하면 표시됨

## 필요한 정보 체크리스트

다음 정보를 Figma에서 확인해주세요:

### 레이아웃
- [ ] 카드 전체 너비
- [ ] 카드 내부 패딩 (상하좌우)
- [ ] 컨테이너 양옆 여백 (padding)
- [ ] 제목과 폼 사이 간격
- [ ] 입력 필드 간 간격 (gap)
- [ ] First Name / Last Name 사이 간격
- [ ] 버튼과 폼 사이 간격

### 색상
- [ ] 배경 그라데이션 색상
- [ ] 카드 배경색
- [ ] 텍스트 색상 (제목, 설명, 라벨)
- [ ] 입력 필드 보더 색상
- [ ] 버튼 배경색
- [ ] 링크 색상

### 타이포그래피
- [ ] 제목 폰트 크기
- [ ] 설명 텍스트 크기
- [ ] 라벨 폰트 크기
- [ ] 입력 필드 폰트 크기
- [ ] 버튼 텍스트 크기

### 기타
- [ ] 카드 border-radius
- [ ] 입력 필드 border-radius
- [ ] 버튼 border-radius

## 정보 제공 방법

다음 중 편한 방법으로 정보를 알려주세요:

1. **스크린샷 + 수치**: Figma에서 측정한 수치를 알려주세요
2. **Figma 링크 공유**: 공유 링크를 주시면 확인 가능합니다
3. **디자인 토큰**: Figma에서 export한 디자인 토큰 정보

## 예시

```
카드 너비: 480px
카드 패딩: 48px (모든 방향)
컨테이너 패딩: 24px
제목-폼 간격: 32px
입력 필드 간격: 20px
First/Last Name 간격: 16px

배경: #667eea → #764ba2 (그라데이션)
카드 배경: #FFFFFF
제목 색상: #1a202c
```

이런 식으로 알려주시면 바로 적용해드립니다!
